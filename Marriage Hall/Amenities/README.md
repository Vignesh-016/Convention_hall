# Amenities Page

This page showcases the amenities and facilities available at Bala Saraswathi Conventional Hall.

## Features

### 1. Hero Section with Booking Form
- Full-width hero section with background image
- Location info and navigation menu
- Breadcrumb navigation (Home > Amenities)
- Interactive booking form with:
  - Date range pickers (From/To)
  - Event type dropdown (Marriage, Reception, etc.)
  - Facilities dropdown (Catering, Decoration, etc.)
  - Submit button

### 2. About/Amenities Section
- Vertical image gallery (6 images with decorative patterns)
- Large heading with decorative elements
- Detailed description text
- Feature badges:
  - Air Conditioned Hall
  - 24hr CCTV Surveillance

### 3. Main Hall Showcase
- Featured venue section with background image
- Main Hall heading and description
- 4-column grid of amenities:
  - Grand stage with elegant backdrop & floral decoration
  - LED lighting with customizable color themes
  - 600-1,000 guests capacity
  - Central air conditioning & premium sound system

### 4. FAQ Section
- Large "Frequently Ask Questions" heading
- Accordion-style FAQ items (4 questions)
- Expand/collapse functionality
- One item expanded by default

### 5. Footer
- Dark maroon background with decorative pattern
- Logo and tagline
- Important Links column
- Contact information
- Email subscription form
- Social media icons
- Copyright text

## Color Palette

- Primary: #780000 (Maroon)
- Secondary: #C08A00 (Gold)
- Accent: #FFDE8A (Light Gold)
- Background: #FFFAEE (Cream)
- Light Background: #FFFDFA
- Text Dark: #303030
- Text Light: #FFFFFF

## Typography

- **League Spartan**: Headings and body text
- **Reem Kufi**: Navigation menu
- **Mea Culpa**: Decorative script text (footer tagline)

## Files

- `amenities.html` - Main HTML structure
- `amenities.css` - Styling for the amenities page
- `amenities.js` - Interactive functionality (FAQ accordion, form validation)

## Usage

The page is accessible from the main navigation menu under "Amenities". It provides a comprehensive overview of the hall's facilities and allows visitors to submit booking inquiries.
